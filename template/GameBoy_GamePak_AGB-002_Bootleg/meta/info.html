GAME BOY Game Pak

AGB-002 (Bootleg)

This project is a template for a bootleg Game Pak/cartridge for the Game Boy Advance (fits bootleg/aftermarket cartridge shell only). It includes a PCB edge connector commonly found on bootlegs (no vias).
